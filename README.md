Spotify Dashboard - Power BI

Overview
This project is an interactive **Spotify Analytics Dashboard** built using **Microsoft Power BI**. The dashboard provides insights into Spotify music data through interactive visualizations, helping users explore trends, artist performance, track popularity, and listening patterns.

----------------------------------

Dashboard Features
- Artist-wise analysis
- Track popularity insights
- Trend analysis using area charts
- Category comparison using bar and column charts
- Distribution analysis with donut charts
- KPI cards displaying important metrics
- Interactive slicers for dynamic filtering

----------------------------------
Tools & Technologies
- **Microsoft Power BI**
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling

----------------------------------
Dashboard Components
The dashboard includes:
- Bar Charts
- Column Charts
- Area Chart
- Donut Charts
- KPI Cards
- Interactive Slicers
- Icons and Custom Visual Elements

----------------------------------
How to Use
1. Download the repository.
2. Open **SpotifyDashboard.pbix** using **Microsoft Power BI Desktop**.
3. Refresh the data (if applicable).
4. Explore the dashboard using the available filters and slicers.

----------------------------------
Key Insights
The dashboard enables users to:
- Identify top-performing artists.
- Analyze music popularity trends.
- Compare different categories of tracks.
- Understand distribution across various metrics.
- Interactively filter and explore the dataset.
